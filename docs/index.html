Multi-Language Sentiment Engine

Bem-vindo à página de demonstração do Multi-Language Sentiment Engine de Gabriel Demetrios Lafis.

Este projeto oferece uma API robusta para análise de sentimento em múltiplos idiomas, detecção de emoções, análise baseada em aspectos e detecção de sarcasmo, utilizando modelos de transformers de última geração.

Arquitetura do Sistema

O diagrama abaixo ilustra a arquitetura modular e escalável do motor de análise de sentimento:

Funcionalidades Principais

  • Análise de Sentimento Multi-idioma
  • Detecção de Emoções
  • Análise de Sentimento Baseada em Aspectos (ABSA)
  • Detecção de Sarcasmo
  • API RESTful com FastAPI
  • Métricas e Monitoramento com Prometheus
  • Cache com Redis para otimização de performance

Para mais detalhes sobre o projeto, incluindo instalação e uso da API, visite o repositório GitHub.
